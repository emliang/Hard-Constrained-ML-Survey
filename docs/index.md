# Reader Guide

This companion site is for understanding the landscape of hard-constrained machine learning.

It is written for readers who want to compare method families, interpret feasibility claims, or decide what kind of method might fit a constrained prediction or generation problem.

## Recommended route

1. Read [Start Here](start-here.md) for the core framing.
2. Use [Taxonomy Overview](taxonomy/overview.md) to see the main branches.
3. Use [Which Method Family?](reader-guide/which-method-family.md) when matching a problem to candidate approaches.
4. Use [How to Read a Hard-Constraint Claim](reader-guide/how-to-read-a-hard-constraint-claim.md) when evaluating papers.
5. Use [Guarantee Terminology](evaluation/guarantee-terminology.md) and [Reporting Dimensions](evaluation/reporting-dimensions.md) to compare evidence.
